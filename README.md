

# Database Schema Design Activity (University + Library)

This repo contains step-by-step relational database schema designs with:
- Tables + relationships (1-N, N-N)
- Primary keys / foreign keys
- Constraints (UNIQUE, CHECK)
- ER diagrams (Mermaid)

## Contents
- `schemas/university/schema.sql`
- `schemas/library/schema.sql`
- `diagrams/ERD.md`

## Activity Objective
Design schemas that reduce redundancy (normalization) and improve integrity using relationships + constraints.
